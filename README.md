## 📖 Problem Statement
Procrastination, cognitive overload, and poor time estimation cause users to miss critical deadlines. Standard to-do list applications do not adapt dynamically when users fall behind; they simply show overdue dates, leading to guilt and further avoidance.

**Deadline Hero AI** solves this by acting as a proactive, interactive companion. When things get tight, users can activate **Rescue Mode** to generate high-efficiency, micro-step salvage plans, while the **AI Coach** and **Dynamic Daily Planner** ensure users stay on track before ever reaching the danger zone.

---

## ⚡ Key Features

### 1. 🎯 AI Task Prioritization
- Evaluates task deadlines, custom priorities, and complexity values.
- Uses Gemini models to dynamically re-rank and arrange tasks into a mathematically optimized sequence.

### 2. 🗂️ AI Task Breakdown
- Breaks down massive, overwhelming tasks into clean, actionable sub-tasks.
- Automatically estimates the hours required for each sub-task block.

### 3. 📅 Daily Planner
- Synthesizes all pending tasks, user-specific productivity styles, and focus goals.
- Compiles an hour-by-hour structured daily schedule directly generated by Gemini.

### 4. 🚨 Rescue Mode
- A tactical feature designed to salvage critical tasks when a deadline is imminent.
- Generates a custom step-by-step salvage plan to execute immediately.

### 5. 💬 Productivity Coach
- An interactive, contextual chatbot trained on the user's focus styles and dynamic task dashboard.
- Offers direct coaching, procrastination mitigation strategies, and time-management tips.

### 6. 🔍 Search & Calendar Navigation
- Easily search through tasks, descriptions, and categories.
- Filter tasks instantly using the integrated date picker.
- Use simple increment (`+1 Day`) or decrement (`-1 Day`) controls to quickly adjust deadlines on the fly.

### 7. 🔒 Multi-user Sandboxing
- High-efficiency session handling utilizing custom workspaces per username.
- Fully isolated workflows, task lists, and conversational history.

## 🛠️ Tech Stack

- **Frontend**: React (v19), Vite, Tailwind CSS (v4), Motion (f.k.a Framer Motion) for animations, Lucide React (iconography), Recharts (data visualizations).
- **Backend**: Node.js, Express.js, TypeScript with hot compilation via `tsx`.
- **Database**: Native file-backed persistent database (`database.json`) with per-user namespacing.
- **AI Engine**: Official `@google/genai` TypeScript SDK utilizing the Gemini 3.5 Flash model.

## 🚀 Setup Instructions

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Deadline-Hero-AI.git
   cd Deadline-Hero-AI
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
   The application will run on `http://localhost:3000`.

5. Build for production:
   ```bash
   npm run build
   npm start
   ```

## 🤖 AI Integrations & Prompts
Deadline Hero AI leverages **Google Gemini** models to deliver contextual, high-value assistance:
- **Prioritization Endpoint**: Analyzes current load and returns a restructured priority order.
- **Breakdown Endpoint**: Synthesizes high-level titles into granular sub-steps with precise time estimates.
- **Rescue Plan Endpoint**: High-urgency context prompt mapping step-by-step action items to the remaining available hours.
- **Interactive Coach**: Maintains conversation history and dynamically references the user's actual dashboard list to provide hyper-personalized productivity feedback.
